О нас

На главную

Наше веб-приложение для бронирования отелей создано с использованием стека технологий Java, JavaScript, HTML, CSS, Maven WebApp и Apache Tomcat.

Мы стремимся предоставить нашим клиентам удобный и надежный сервис для поиска и бронирования отелей по всему миру.

Наше приложение имеет несколько слоев: слой сущностей (entity) для описания таблиц из базы данных; слой сервисов (service) для работы с базой данных; слой сервлетов (servlet) для создания response и request сервлетов.

Мы постоянно работаем над улучшением нашего сервиса и расширением его функциональности. Спасибо за выбор нашего приложения!
